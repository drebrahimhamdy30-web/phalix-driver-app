# Phalix Driver — تطبيق سائق التوصيل

تطبيق أندرويد لسائقي التوصيل، هدفه الأساسي **استقبال إشعارات الطلبات الجديدة بصوت عالٍ وموثوق** (عبر Firebase Cloud Messaging) — يعمل حتى والتطبيق مقفول.

## المميزات
- تسجيل دخول السائق (عبر n8n).
- استقبال إشعار عند إسناد طلب جديد، بصوت تنبيه مميّز، حتى والتطبيق مغلق.
- شاشة تعرض طلبات السائق النشطة.

## البناء
يُبنى تلقائيًا عبر GitHub Actions (`.github/workflows/build-apk.yml`).
ملف الـ APK يظهر في نتائج الـ Actions (Artifacts).

## التقنيات
Flutter · Firebase Cloud Messaging · Supabase (REST) · n8n
